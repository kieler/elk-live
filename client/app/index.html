ELK Graph Viewers

Automatic layout algorithms compute layouts for diagrams. Never tried them? Well, here's your chance. Choose one of the thingies below and start playing!

ELKT Editor

An interactive diagram editor based on the Eclipse Layout Kernel's textual graph format. Basically, build your own graphs and see what automatic layout does with them.

ELKT Editor

JSON Editor

An interactive diagram editor based on the Eclipse Layout Kernel's JSON format. Another variation on the build-your-own-graphs theme, but with JSON, which makes everything better.

JSON Editor

Model Browser

An interactive browser for the Eclipse Layout Kernel's repository of test models. See what automatic layout does to diagrams that we have created for you (well, mainly for us, but you may look at them, too).

Graph Browser
